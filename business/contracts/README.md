## 業務委託契約

- [業務委託契約｜社長のための労働相談マニュアル](https://www.mykomon.biz/keitai/rodo/rodo_itaku.html)

- 法律行為以外の事務を行うことを受諾した者が自分の責任・管理の下で、その事務処理を行うことを約束する契約（民法656条）
- 事務の処理が目的となり、労務の提供そのものは目的とはない
- 注文者が受託業者の労働者を直接指揮命令できない

# [git-stash](https://git-scm.com/docs/git-stash)

## 最後の `stash` を戻す

~~~bash 
$ git stash pop
~~~

## 退避するファイルを選ぶ

- `退避したくない` ファイルを `git add` する
- その後で `git stash` する

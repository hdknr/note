# Apple Silicon(M1)

## arch コマンド

~~~zsh
% uname -m
arm64
~~~

~~~zsh
% uname -m
% arch -x86_64 zsh
% uname -m
x86_64
~~~ 

## pyenv

~~~zsh
% export LDFLAGS="-L/usr/local/lib"; export CPPFLAGS="-I/user/local/include"
% arch -x86_64 pyenv install 3.9.9
~~~

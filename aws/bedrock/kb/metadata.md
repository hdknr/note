# メタデータフィルタリング

## メターデータファイル

- ` ソースファイル``.metadata.json `
- 10KB 上限

メタデータファイル例:

```json
{
  "metadataAttributes": {
    "tag": "project EVE",
    "year": 2016,
    "team": "ninjas"
  }
}
```

# アクション

- ルールで定義された条件に一致したときにウェブリクエストをどう処理するか

AWS WAFアクション:

| アクション | 対応                                         |
| ---------- | -------------------------------------------- |
| ALLOW      | そのリクエストを許可                         |
| BLOCK      | そのリクエストを拒否                         |
| COUNT      | カウントとして検知をし、リクエストに対して許可 |

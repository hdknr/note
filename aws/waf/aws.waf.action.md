# アクション

AWS WAFアクション:

| アクション | 対応                                         |
| ---------- | -------------------------------------------- |
| ALLOW      | そのリクエストを許可                         |
| BLOCK      | そのリクエストを拒否                         |
| COUNT      | ウントとして検知をし、リクエストに対して許可 |

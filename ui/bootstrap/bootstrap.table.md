## カラム幅

- `colgroup`, `col` で指定

~~~html
<table class="table">
  <colgroup>
   <col class="col-md-2">
   <col class="col-md-5">
   <col class="col-md-5">
  </colgroup>
  <thead>
    <tr>
      <td>Day</td>
      <td>West</td>
      <td>East</td>
    </tr>
  </thead>
  <tbody>
    <tr>......
  </tbody>
</table>
~~~

# WSL

## インストール

- `スタート` > `設定` > `アプリ`
- `プログラムと機能` > `コントロールパネル`
- `Windowsの機能の有効化または無効化`
- ツリーから `Windows Subsystem for Linux` を選択して再起動

![](windows-wsl.png)

## "Ubuntu"をMicrosoft Storeから入れる

- `スタート`> `Microsoft Store`
- 検索 "Linux"
- `Ubuntu` ダウンロード(214.4MB)
- `インストール` 

![](windows-ubuntu.png)
